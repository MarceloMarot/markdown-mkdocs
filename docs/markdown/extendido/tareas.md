# Listas de Tareas

Las listas de tareas son una variante de las listas no ordenadas. 
Agregan una secuencia tras cada guión:

- `[x]` para marcar el ítem; 
- `[ ]` para desmarcarlo.

Ejemplo:

<div class="grid" markdown>


``` md 
- [x] Paso 1
- [ ] Paso 2
- [ ] Paso 3
- [x] Opcional
```


- [x] Paso 1
- [ ] Paso 2
- [ ] Paso 3
- [x] Opcional

</div>
