# Privacidad de Datos



## Plugin *privacy*

``` yaml title="Plugin privacy"
# "mkdocs.yml"
plugins:
  - privacy
```

Esto ayuda a cumplir con la Regulacion General de Protección de Datos de la Unión Europea.
